# BOJ 1684 같은 나머지
## 문제 요약
n개의 정수로 된 수열이 있을 때, 모든 정수를 한 정수 D로 나눴을 때 나머지가 같아지는 경우가 있다. 수열에 따라서 이러한 정수 D가 여러 개 존재할 수 있다.
n개의 정수로 된 수열이 주어졌을 때, 가장 큰 D를 구하면 된다.

++) n은 1보다 크거나 같고 1000보다 작거나 같다. 수열을 이루는 정수들은 절대값이 1,000,000을 넘지 않는다.

## Naive한 접근
식을 써보기 전까지 접근하기 어려워 Naive한 접근은 건너뛴다.

위의 사진에서 두 수의 나머지가 같도록하는 D에 대해 ③을 유도할 수 있었다. 이 식을 통해 D가 가장 커지기 위해선 D가 n₁ - n₂이면 된다는 것을 알 수 있다. 수가 여러 개일 때는 각 수들 사이의 차이가 여러 개 생길 수 있다.
두 수의 경우에서 두 수를 각각 n₁ - n₂의 약수로 나누어도 나머지가 같기에 수가 여러 개 있을 때는 각 수의 차이들의 최대공약수를 D로 설정하면 된다는 것을 알 수 있다.

## 구현 포인트
차이값을 저장하는 과정에서 리스트 컴프리헨션과 조건문을 함께 사용하여 불필요한 연산을 제거하고 코드를 단순화하였다. 정수론 문제이기에 구현적으로 어려운 부분은 크게 없었다.

# BOJ 10844번 쉬운 계단 

## 문제 요약
인접한 모든 자리의 차이가 1인 수를 계단 수라고 한다.  
N이 주어질 때, 길이가 N인 계단 수가 총 몇 개 있는지 구하면 된다.

++) N은 1보다 크거나 같고, 100보다 작거나 같은 자연수이다. 0으로 시작하는 수는 계단수가 아니다.

## Naive한 접근
특정 자리수의 수의 뒤에 붙을 수 있는 수가 정해져 있으므로 점화식을 세워 바텀업 방식으로 dp 테이블을 채운다. 정답을 1,000,000,000으로 나눈 나머지를 출력한다

## Attempts
## Attempt 1 - WA
dp[자리수][끝자리 숫자]로 구성한 1-base 2차원 dp 테이블을 만들었다. 이중 포문을 돌리며 현재 자리수와 끝 자리 숫자에서 이전 자리수의 끝자리 숫자 - 1, 끝자리 숫자 + 1의 값을 더하고 모듈러스 연산을 적용한다. 
출력은 sum(dp[N])으로 하였는데 이 값이 1,000,000,000을 넘어갈 수 있어 모듈러스 연산을 적용해야하는데 그렇지 않아서 WA를 받았다.  

## Attempt 2 - Accepted Solution
Attempt 1의 출력에 모듈러스 연산을 적용하여 AC를 받았다.

## 구현 포인트
자리수 부분을 1-base로 하여 헷갈리지 않도록 하였다. 모듈러스 연산은 부등식이 아닌 경우 로직 내의 모든 연산 뒤에 추가하는 것이 안전하다.
